## Internet Tehnologije 2025 - App za Iznajmljivanje Knjiga

### Project Description
This application is a web-based platform for book rentals developed for the Internet Tehnologije 2025 course. The system supports user authentication, book catalog management via external API integrations, and rental tracking.

### Tech Stack
- Frontend: React.js (Vite), Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB / PostgreSQL
- Authentication: JSON Web Tokens (JWT)
- External APIs: Google Books / Open Library

### Branch Management
The project utilizes a feature-branch workflow to manage collaborative development:
- main: Stable production-ready code.
- develop: Integration branch for ongoing development.
- feature/eksterni-api: Implementation of third-party API services.
- registracijaVuk: Development of user registration and validation logic.
